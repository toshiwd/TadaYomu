# TadaYomu

TadaYomu is a simple Android application for reading novels.

## Setup

- **JDK**: 17
- **Android Studio**: Iguana | 2023.2.1 or later
- **Android SDK**: 34

## Build

To build the project, run the following command:

```bash
./gradlew assembleDebug
```

## Firebase

This project does not currently require Firebase. If it becomes necessary in the future, the setup instructions will be added here.
